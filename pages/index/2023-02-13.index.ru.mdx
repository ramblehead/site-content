# Мой цифровой путь

Меня зовут Виктор Рыбынок. Я старший инженер-программист, занимаюсь разработкой
программного обеспечения для систем квантового распределения ключей (Quantum Key
Distribution, QKD).

Этот сайт представляет собой коллекцию блог-публикаций и заметок, а также место
где я могу удерживать свои мысли в организованной форме, с возможностью поиска и
открытыми остальному миру&nbsp;— моё пространство в
[неосфере](https://ru.wikipedia.org/wiki/Ноосфера).

Также, это тестовая платформа для экспериментов с методами разработки и
развёртывания веб-приложений, и документирования небольших открытий, которые мне
представляются интересными.
