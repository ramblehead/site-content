### BLTouch Leveling Sensor wiring

We are currently using this sensor:<br />
https://www.ebay.co.uk/itm/204184579333

#### 3 Pins

| Colour | Function |
| ------ | -------- |
| Green  | GND      |
| Red    | +5V      |
| Orange | Signal   |

#### 2 Pins

| Colour | Function |
| ------ | -------- |
| Black  | GND      |
| White  | Z Min    |

### Related G-codes and Commonly Used commands

`G29 P1` initial mesh populate<br /> `G29 T` output mesh topology

- [M421 - Set Mesh Value](https://marlinfw.org/docs/gcode/M421.html)
